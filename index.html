Fan Control System

I designed and built a open loop fan system using an Elegoo Uno, L293D motor driver, DC motor, push buttons and indicator LEDs. The system uses latching logic to start and stop the fan via operator controls. The project demonstrates digital inputs, digital outputs, motor control and basic automation principles.


          const int startButton = 8;
          const int stopButton = 9;
          
          const int greenLED = 5;
          const int redLED = 6;
          
          const int motorPin = 7;
          
          bool fanRunning = false;
          
          void setup()
          {
            pinMode(startButton, INPUT_PULLUP);
            pinMode(stopButton, INPUT_PULLUP);
          
            pinMode(greenLED, OUTPUT);
            pinMode(redLED, OUTPUT);
          
            pinMode(motorPin, OUTPUT);
          
            digitalWrite(redLED, HIGH);
          }
          
          void loop()
          {
            if (digitalRead(startButton) == LOW)
            {
              fanRunning = true;
            }
            if (digitalRead(stopButton) == LOW)
            {
              fanRunning = false;
            }
            if (fanRunning)
            {
              digitalWrite(greenLED, HIGH);
              digitalWrite(redLED, LOW);
              digitalWrite(motorPin, HIGH);
            }
            else
            {
              digitalWrite(greenLED, LOW);
              digitalWrite(redLED, HIGH);
              digitalWrite(motorPin, LOW);
            }
          }

Through this project I learned how to use digital inputs and outputs, control a DC motor using an L293D motor driver, implement latching logic, troubleshoot wiring issues and interpret breadboard layouts and pin configurations.

Smart Security Light Controller with Manual Override and Fault Detection

When the LDR detects darkness, such as when the sensor is covered by a hand, the system enters its active operating state. If the PIR motion sensor detects movement while the environment is dark, the green LED illuminates to simulate a security light. The light remains on for a preset period before automatically switching off if no further motion is detected, demonstrating the use of timers and automation logic.

The Smart Security Light Controller is designed to operate only in low-light conditions. An LDR continuously measures the surrounding light level. When the area is brightly lit, such as when a flashlight is shone directly onto the sensor, the controller prevents the security light from turning on even if motion is detected. This simulates the behaviour of commercial outdoor security lighting, conserving energy by only operating when illumination is required.

A manual override function has been implemented using a push button. Pressing the button switches the controller into Manual Mode, indicated by a blue LED. In this mode, the security light remains on continuously regardless of the PIR sensor or ambient light level. Pressing the button again returns the controller to Automatic Mode, allowing the motion and light sensors to resume normal operation.

The controller also includes a basic fault detection feature. During operation, disconnecting the ground wire from the LDR simulates a sensor failure. When this occurs, the controller detects the abnormal sensor reading, illuminates the red fault LED and indicates that a fault has been detected. Once the ground wire is reconnected, the sensor returns to normal operation, the red fault indicator switches off and the controller resumes automatic operation. This demonstrates simple fault monitoring and recovery, similar to features found in industrial automation and control systems.


          // Pins
          const int pirPin = 2;
          const int buttonPin = 3;
          
          const int greenLed = 8;
          const int blueLed = 9;
          const int redLed = 10;
          
          const int ldrPin = A0;
          
          // Settings
          const int darkThreshold = 550;          
          const unsigned long lightTime = 10000;  
          
          // Variables
          bool manualMode = false;
          bool lastButtonState = HIGH;
          bool lastMotion = LOW;
          
          unsigned long lightTimer = 0;
          
          void setup()
          {
            pinMode(pirPin, INPUT);
            pinMode(buttonPin, INPUT_PULLUP);
          
            pinMode(greenLed, OUTPUT);
            pinMode(blueLed, OUTPUT);
            pinMode(redLed, OUTPUT);
          
            Serial.begin(9600);
          }
          
          void loop()
          {
            // Read Sensors
            int motion = digitalRead(pirPin);
            int lightLevel = analogRead(ldrPin);
          
            // Manaual Mode Button
            bool currentButton = digitalRead(buttonPin);
          
            if (lastButtonState == HIGH && currentButton == LOW)
            {
              manualMode = !manualMode;
              delay(50); 
            }
          
            lastButtonState = currentButton;
          
            // Motion Timer
            if (motion == HIGH && lastMotion == LOW && lightLevel < darkThreshold)
            {
              lightTimer = millis();
            }
          
            lastMotion = motion;
          
            // Fault Detection
            bool fault = false;
          
            if (lightLevel <= 5 || lightLevel >= 1018)
            {
              fault = true;
            }
          
            digitalWrite(redLed, fault);
          
            // Lighting Control
            if (manualMode)
            {
              digitalWrite(blueLed, HIGH);
              digitalWrite(greenLed, HIGH);
            }
            else
            {
              digitalWrite(blueLed, LOW);
          
              if (!fault)
              {
                if (millis() - lightTimer < lightTime)
                {
                  digitalWrite(greenLed, HIGH);
                }
                else
                {
                  digitalWrite(greenLed, LOW);
                }
              }
              else
              {
                digitalWrite(greenLed, LOW);
              }
            }
          
          // Serial Monitor
            Serial.print("Mode: ");
            Serial.print(manualMode ? "MANUAL" : "AUTO");
          
            Serial.print(" | Motion: ");
            Serial.print(motion ? "YES" : "NO");
          
            Serial.print(" | Light Level: ");
            Serial.print(lightLevel);
          
            Serial.print(" | ");
            if (lightLevel < darkThreshold)
              Serial.print("DARK");
            else
              Serial.print("BRIGHT");
          
            Serial.print(" | Green LED: ");
            Serial.print(digitalRead(greenLed) ? "ON" : "OFF");
          
            Serial.print(" | Fault: ");
            Serial.println(fault ? "YES" : "NO");
          }
